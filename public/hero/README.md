# Hero Slide Images

Place your 4 architectural rendering images in this folder with the exact filenames below.

| Filename    | Slide Text              | Description                           |
|-------------|-------------------------|---------------------------------------|
| slide1.jpg  | Modern Homes            | Clean modern villa exterior render    |
| slide2.jpg  | Affordable Bungalows    | Single-storey bungalow rendering      |
| slide3.jpg  | Luxury Apartments       | Multi-storey apartment block render   |
| slide4.jpg  | Contemporary Layouts    | Sleek contemporary house rendering    |

## Specifications
- Dimensions : 1440 × 900 px minimum (16:9 aspect ratio)
- Format     : JPG or WebP (WebP preferred — smaller file size)
- File size  : ≤ 300 KB per image
- Color space: sRGB

## To update slide text or filenames
Edit `src/config/publicSlides.js` — the carousel engine reads from there.
